# Support

## Community of Practice (CoP)

Please join the [Chap Community of Practice (CoP)](https://community.dhis2.org/c/development/chap/84) where you can get answers to your questions from other members of the community. 

## Issues/Bugs

If you find any bugs or issues when using this code base, we appreciate it if you file a [GitHub bug report here](https://github.com/dhis2-chap/chap-core/issues/new)

## Email

For any other questions, including collaboration requests or similar, please send us an email at [chap[at]dhis2.org](mailto:chap@dhis2.org).
