# DHIS2 Modeling - Guides & Tutorials

In this section you will find various guides to help you get started as a user or contributor to the DHIS2 Chap Modeling Platform.

## Implement Chap for DHIS2

To install and integrate Chap with a DHIS2 instance, a **system admin** should install the chap-core backend and configure the DHIS2 Modelling App:

- [Setup the Chap Core backend engine](https://dhis2-chap.github.io/chap-core/modeling-app/installation.html)
- [Configuring the DHIS2 Modeling App to work with Chap](https://dhis2-chap.github.io/chap-core/modeling-app/running-chap-on-server.html)

## Model development

If you are a model developer and want to develop or integrate your own models to work with Chap, we provide detailed documentation on how to get started:

- [Overview for model developers](https://dhis2-chap.github.io/chap-core/external_models/index.html)

## Run Chap from the command line

For modelers or researchers more used to command line interfaces, Chap is also available as a standalone tool:

- [Guide for getting started with Chap on the commandline](https://dhis2-chap.github.io/chap-core/chap-cli/chap-core-cli-setup.html)
